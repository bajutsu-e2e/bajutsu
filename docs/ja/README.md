# Bajutsu ドキュメント

このディレクトリは Bajutsu のドキュメントサイトのソースです。検索とナビゲーションと英語／日本語の切り替えが付いた、閲覧できる公開版は **<https://bajutsu-e2e.github.io/bajutsu/ja/>** にあります。

GitHub でソースを読むなら、機能ごとのリファレンスを推奨読書順に並べた [overview.md](overview.md) から始めるか、[Getting started のチュートリアル](getting-started.md)へ直接進んでください。[用語集](glossary.md)は、ドメインの用語を一語ずつ定義します。自動生成の Python API リファレンスは [`api/`](../api/index.md) にあります。
